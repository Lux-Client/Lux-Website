Analytics Opt-Out

Your privacy is in your hands.

MCLC uses anonymized analytics data to improve the stability and performance of the launcher. No personal data such as passwords or private files are transmitted.

? How can I opt out?

You can disable the collection of analytics data at any time in the launcher settings:

  1. Open MCLC.
  2. Navigate to the Settings tab.
  3. Find the Integration section.
  4. Disable the Discord Rich Presence option or similar telemetry features (if available).

Technical Note

Note: Certain basic technical data (such as operating system version) may still be necessary for the operation of the launcher.
